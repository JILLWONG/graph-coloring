#### 分析

前几个算例的速度较快，可达300k/s，后面的速度减慢至50k/s，原因分析如下：仇人表采用二维数组，未记录邻域动作，每次寻找最优delt值时需要遍历仇人表，搜索时间复杂度为O（N^2），随着N的增加，每次迭代耗时也增加。

#### 遗留问题

遗传算法正在实现中，经分析其时间复杂度较高，打算先通过记录邻域动作的方法提速后，再加入遗传算法。
